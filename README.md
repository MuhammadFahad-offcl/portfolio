# Muhammad Fahad Portfolio 

This project recreates the public portfolio at [muhammadfahad-portfolio.lovable.app](https://muhammadfahad-portfolio.lovable.app) in a GitHub-ready React + TypeScript + Vite structure.

## Stack

- React
- TypeScript
- Vite

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project structure

```text
.
|-- index.html
|-- package.json
|-- src
|   |-- App.tsx
|   |-- components
|   |   `-- LovableLogo.tsx
|   |-- index.css
|   `-- main.tsx
|-- tsconfig.json
|-- tsconfig.node.json
`-- vite.config.ts
```

## Included sections

- Fixed header with desktop and mobile navigation
- Cinematic hero section with showcase card
- About, Skills, Projects, Certifications, Contact, and Footer sections
- Responsive styling suitable for GitHub upload and deployment
